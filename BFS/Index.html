Aim

To search a tree or graph data structure for a node that meets a set of criteria using BFS

Theory

Breadth-first search is a graph traversal algorithm that starts traversing the graph from the root node and explores all the neighboring nodes. Then, it selects the nearest node and explores all the unexplored nodes. While using BFS for traversal, any node in the graph can be considered as the root node.

Procedure

  1. Create an empty queue data structure. This queue will hold the nodes to be explored.
  2. Mark the starting node as visited and enqueue it into the queue.
  3. While the queue is not empty, repeat steps 5-7.
  4. Dequeue a node from the front of the queue. This node will be the current node for exploration.
  5. Process the current node (print it, store it, or perform any desired operation).
  6. Enqueue all the unvisited neighbors of the current node into the queue and mark them as visited.
  7. Repeat steps 5-7 until the queue becomes empty.
  8. If there are still unvisited nodes in the graph, select one of them as the new starting node and repeat steps 3-8.
  9. Once all nodes have been visited or processed, the BFS traversal is complete.

Simulation

Python Code

                        
                            

                            from collections import deque
                            def bfs(graph, start):
                                visited = set()                 # Set to keep track of visited nodes
                                queue = deque([start])          # Create a queue and enqueue the starting node
                
                                while queue:
                                    node = queue.popleft()      # Dequeue a node from the front of the queue
                                    print(node)                 # Process the current node (you can replace this with your desired operation)
                
                                    if node not in visited:
                                        visited.add(node)       # Mark the node as visited
                
                                        # Enqueue all the unvisited neighbors of the current node into the queue
                                        for neighbor in graph[node]:
                                            if neighbor not in visited:
                                                queue.append(neighbor)
                
                            # Example usage
                            # Define the graph as an adjacency list
                            graph = {
                                'A': ['B', 'C'],
                                'B': ['D', 'E'],
                                'C': ['F'],
                                'D': [],
                                'E': ['F'],
                                'F': []
                            }
                
                            # Call the BFS function with the graph and the starting node
                            bfs(graph, 'A')

Result

Hence we can perform searching of all the vertices of a graph using BFS.
